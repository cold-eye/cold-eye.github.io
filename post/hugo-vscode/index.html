hugo 博客编辑器

  vscode 即 visual studio code，下载Markdown Preview Enhanced插件，即可实现边写博客边预览效果。

1、安装插件

  • 打开 VSCode 之后点击左侧的扩展（快捷键shift + command + X），在搜索框搜索Markdown Preview Enhanced插件，下载之后重启VSCode。

2、打开编辑

  • 重启 vscode 之后，打开一篇空白 .md 文件，在文本上 control + 鼠标左击，即可在右侧打开插件。
