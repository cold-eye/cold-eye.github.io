摄氏温度转华氏温度转化公式：华氏 = （摄氏 X 1.8） + 32

一、python

1
2
3
4

#!/usr/bin/python
celsius = 10
fahrenheit = (celsius * 1.8) + 32
print(f'{celsius} 摄氏温度转为华氏温度为 {fahrenheit}')

二、rust

1
2
3
4
5

fn main() {
	let celsius : f64 = 10.0;
	let fahrenheit : f64 = (celsius * 1.8) + 32.0; // 这里的 32.0 要注意，与python不同。rust运算时，明确数据类型要相同。
	println!("{} 摄氏温度转为华氏温度为 {}", celsius, fahrenheit) // 打印的时候会发现，10.0打印出来会变成10
}
